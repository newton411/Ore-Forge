# Ore Forge Frontend

This frontend is a playable prototype for the Supra L1 idle/clicker mining game.

## Gameplay loop
- Tap the mining button to gain ORE.
- Spend ORE on pickaxe and drill upgrades.
- Purchase temporary boosts for multiplier effects.
- Connect StarKey to prepare for on-chain integration.

## Wallet integration
- This prototype uses the official StarKey wallet integration path via window.starkey.supra.
- No alternative wallet providers are included.
